# *Proyecto base de datos: MARKETZET*
### _Sitio web con node js/ Html5/ JavaScript/ Css / MySql_
---

## Descripción

Este proyecto tiene como objetivo desarrollar una base de datos integral para la gestión de usuarios, productos, compras y más. Está diseñado para ser escalable, eficiente y fácil de usar, con una estructura clara y bien documentada.

## Características

- **Gestión de Usuarios:** Registro, autenticación y autorización de usuarios.
- **Gestión de Productos:** Añadir, editar y eliminar productos con categorías asociadas.
- **Carrito de Compras:** Funcionalidad completa de carrito de compras.
- **Procesamiento de Compras:** Gestión y seguimiento de pedidos y pagos.
- **Comentarios:** Sistema de comentarios para productos.

## Estructura de la base de datos

El proyecto está dividido en varios módulos para facilitar su mantenimiento y escalabilidad:

- `usuarios/`: Manejo de la información de usuarios.
- `productos/`: Gestión de productos y categorías.
- `carrito/`: Funcionalidad del carrito de compras.
- `compras/`: Procesamiento y gestión de compras.
- `comentarios/`: Sistema de comentarios para productos.

Recuerde que siempre puede revisar el Manual Técnico para una mejor comprensión del proyecto.
